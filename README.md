# JammyJams

JammyJams is an online React component editor and renderer featuring a broad library of templates designed for developers to enhance their React designs.

## Features

- **Component Editor**: Edit and preview React components directly in your browser.
- **Template Library**: Access a wide range of pre-built templates to kickstart your projects.
- **Code Transformation**: Convert regular HTML, CSS, and JavaScript into functional React code.
- **Export Options**: Copy generated React code to your clipboard or export it as a ZIP file for preview in React Player.

## Usage

1. Visit [RicoSuaveDev.github.io/JammyJams](https://RicoSuaveDev.github.io/JammyJams) to access the application.
2. Select a template from the library or start with a blank component.
3. Edit the HTML, CSS, and JavaScript in the editor.
4. Preview the rendered component in real-time.
5. Copy the generated React code or export it as a ZIP for further use.

## Getting Started

No installation is required. Simply navigate to the URL and start creating.
